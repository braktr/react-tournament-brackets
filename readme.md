# React Tournament Brackets (Enhanced Fork)

A fork of the original [react-tournament-brackets](https://github.com/Shenato/react-tournament-brackets) with bug fixes and UX improvements.

---

## 🙏 Credits

This project is based on the original work by the contributors of  
https://github.com/Shenato/react-tournament-brackets

All core functionality, architecture, and design belong to the original authors.

---

## ✨ Changes in this fork

This fork introduces the following improvements:

- 🐛 Fix: Connector component incorrectly updating URL
- 🐛 Fix: SVG anchor click not working properly on mobile devices
- ⚙️ Enhancement: `onMatchClick` is now configurable from match context
- 🎨 UX Improvement: Enhanced "Match Details" button (visual updates)

---

## 🎯 Purpose of this fork

This fork was created to improve mobile usability, fix interaction issues, and allow more flexible match handling for real-world application use cases.
